# SAMCO Project Controls Hub — Assets

## Logo Setup

Place your SAMCO logo in this folder:

- **Recommended:** `logo.svg` (scalable, looks best)
- **Alternative:** `logo.png` (will also be detected)

The logo will automatically appear in the top-left corner of the dashboard.

## Brand Colors Used
- Primary: `#00d4aa` (Teal)
- Secondary: `#00a8e8` (Blue)
- Success: `#00d4aa` (Green)
- Warning: `#f59e0b` (Amber)
- Danger: `#ef4444` (Red)
- Background: `#0d1117` (Dark)
